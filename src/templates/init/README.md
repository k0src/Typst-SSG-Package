# {{PROJECT_NAME}}

A static site built with [Typst Static Site Generator](https://github.com/k0src/Typst-SSG).

## Development

```bash
# Start development server
npm run dev

# Build for production
npm run build
```

## Project Structure

- `pages/` - Typst content files
- `assets/` - Static assets (e.g., images)
- `build/` - Generated static site

- [Typst Documentation](https://typst.app/docs)
